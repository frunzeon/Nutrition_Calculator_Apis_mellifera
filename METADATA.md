# Honey Bee Nutrition Calculator — Metadata

## Sample Groups

| Variable       | Category      | Description |
|----------------|--------------|------------|
| Sample_Group   | 5-day-old     | Bees fed for 5 days post-emergence under cage conditions |
| Sample_Group   | New bees      | Newly emerged workers (<24 h old) |
| Sample_Group   | Nurses        | Worker bees feeding larvae |
| Sample_Group   | Foragers      | Worker bees carrying pollen at hive entrance |

## Diet Codes

| Code   | Description |
|--------|------------|
| BS     | Beebread + 30% sugar syrup |
| B      | Beebread |
| S      | 30% sugar syrup |
| Ah     | Acacia honey |
| Sh     | Sugar honey |
| S-Ala  | 30% sugar syrup + alanine |
| S-Arg  | 30% sugar syrup + arginine |

## Metabolic Score

Continuous variable:
- High → BS-like condition  
- Low → S-like condition

## Visualization

- Red → BS condition  
- Blue → S condition
